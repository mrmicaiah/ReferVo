# ReferVo

**Connect. Refer. Earn.**

A mobile app that lets businesses send qualified leads to trusted partners and earn money when those leads convert. A referral marketplace for professionals.

## Website

This repository hosts the ReferVo marketing website at [refervo.com](https://refervo.com).

## Pages

- **Home** (`/`) - Hero, features, how it works, app store download
- **Privacy Policy** (`/privacy`) - Data collection, usage, and rights
- **Terms of Service** (`/terms`) - User agreements and liability
- **Support** (`/support`) - FAQ and help resources
- **Contact** (`/contact`) - Contact form and email addresses
